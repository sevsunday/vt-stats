    About VT Stats

    VT Stats is a static dashboard for Battlezone: Combat Commander match statistics, built on top of VT's protobuf-based stats collector.

    Source repository
    sevsunday/vt-stats
    Dashboard, processing pipeline, and theme system.
    Stats collector (VT's repo)
    VTrider/statsgate
    In-game client that captures match data and writes the .binpb sessions this dashboard consumes.
    Multiplayer session API
    bz2api.js
    JavaScript library for querying live Battlezone 2: Combat Commander multiplayer session data.
